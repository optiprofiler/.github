### üéØ Our Mission

The **OptiProfiler** organization is dedicated to providing reliable, reproducible, and easy-to-use tools for benchmarking numerical optimization algorithms. We bridge the gap between algorithm development and performance analysis by offering a standardized benchmarking environment across different programming languages and systems.

---

### üöÄ Core Repository

The heart of our ecosystem is the **OptiProfiler** package, available for both **MATLAB** and **Python**.

*   [**optiprofiler**](https://github.com/optiprofiler/optiprofiler): The main benchmarking framework. It allows users to run solvers on large sets of problems, apply various "features" (noise, rotations, quantization), and automatically generate professional performance, data, and log-ratio profiles. Offical website: [optprof.com](https://optprof.com)

### üìö Problem Libraries

We provide interfaces to various problem libraries, allowing `optiprofiler` to access a diverse range of test problems, primarily focusing on the variants of the classic CUTEst collections.

| Repository | Language | Description |
| :--- | :--- | :--- |
| [**s2mpj_matlab**](https://github.com/optiprofiler/s2mpj_matlab) | MATLAB | MATLAB translation of CUTEst, originally sourced from [**S2MPJ**](https://github.com/GrattonToint/S2MPJ) |
| [**s2mpj_python**](https://github.com/optiprofiler/s2mpj_python) | Python | Python translation of CUTEst, originally sourced from [**S2MPJ**](https://github.com/GrattonToint/S2MPJ) |
| [**matcutest**](https://github.com/optiprofiler/matcutest) | MATLAB | Interface to CUTEst for MATLAB, originally sourced from [**MatCUTEst**](https://github.com/matcutest) |
| [**pycutest**](https://github.com/optiprofiler/pycutest) | Python | Interface to CUTEst for Python, originally sourced from [**PyCUTEst**](https://github.com/jfowkes/pycutest) |
